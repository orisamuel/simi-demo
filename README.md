# סימי — מסדרת את כל המשימות

דמו של מערכת ניהול משימות למשרד פרסום, מבית [42 Creative](https://42creative.co.il/).

**דמו חי:** https://orisamuel.github.io/simi-demo/

## מה יש בדמו

- מסלולי אישורים דינמיים לפי סוג תוצר: עבודה ← אישור מחלקה (קריאייטיב / סטודיו / וידאו) ← אישור שותף/ה ← ניהול לקוח
- מנוע כללים חכם: לקוח חדש עובר סבב מלא, טקסט רץ מדלג ישר לניהול לקוח, משימה גדולה מחייבת שותף, תיקון קטן חוזר ישירות — עם הסבר "למה המסלול הזה" על כל משימה
- סבבי תיקונים: החזרה לעובד/ת (או אחר/ת, או לשיבוץ), "אחרי התיקון — ישר אליי", הערות לקוח מכל שלב, גרסאות עם "מה טופל"
- לוחות: המשימות שלי · ממתין לאישורי · לוח לקוחות · ניהול עבודה עם שיבוץ בגרירה · קנבן
- פאנל ניהול: הגדרת מנהל/ת קריאייטיב ושותף/ה לכל לקוח, הרשאות, עריכת כללים ומסלולים

## איך משחקים

מחליפים משתמש בכפתור למעלה כדי לחוות את המערכת מכל תפקיד. כל הנתונים מדומים ונשמרים ב-localStorage של הדפדפן בלבד — "איפוס הדמו" מחזיר להתחלה.

אפשר גם לקפוץ ישירות למצב מסוים: `?user=u_omer&view=approvals&task=q8`
