# סימי — מסדרת את כל המשימות

דמו של מערכת ניהול משימות למשרד פרסום, מבית [42 Creative](https://42creative.co.il/).

**דמו חי:** https://orisamuel.github.io/simi-demo/

## מה יש בדמו

- מסלולי אישורים דינמיים לפי סוג תוצר, כולל אישורי ביניים (קופי עובר אישור תוכן לפני שממשיכים לעיצוב) — ודדליין נפרד לכל שלב עבודה
- מנוע כללים חכם: לקוח חדש עובר סבב מלא, טקסט רץ מדלג ישר לניהול לקוח, משימה גדולה מחייבת שותף, תיקון קטן חוזר ישירות — עם הסבר "למה המסלול הזה" על כל משימה, ועורך כללים עם שילוב תנאים
- סבבי תיקונים: החזרה לכל שלב עבודה (או עובד/ת אחר/ת, או לשיבוץ), "אחרי התיקון — ישר אליי", הערות לקוח מכל שלב, גרסאות עם "מה טופל"
- ניגוד עניינים: מי שביצע את העבודה לא מאשר את עצמו — האישור מוסלם אוטומטית; ממלא/ת מקום לחופשות
- פרויקטים שמקבצים משימות קשורות, מרכז התראות פנימי, ועומס משוקלל לפי גודל משימה בניהול עבודה
- לוחות: המשימות שלי · ממתין לאישורי · לוח לקוחות · ניהול עבודה עם שיבוץ בגרירה · קנבן עם חיפוש
- פאנל ניהול: הגדרת מנהל/ת קריאייטיב ושותף/ה לכל לקוח, הרשאות וממלאי מקום, עריכת כללים ומסלולים

## איך משחקים

מחליפים משתמש בכפתור למעלה כדי לחוות את המערכת מכל תפקיד. כל הנתונים מדומים ונשמרים ב-localStorage של הדפדפן בלבד — "איפוס הדמו" מחזיר להתחלה.

אפשר גם לקפוץ ישירות למצב מסוים: `?user=u_omer&view=approvals&task=q8`
